Automatic Question Generator for Cognitive Bot Evaluation

Internship Project | Python, NLP, NLTK, Stanford NER

  • Developed an innovative question generation system aimed at automating the evaluation of cognitive bots by dynamically creating relevant questions from textual content.
  • Implemented advanced NLP techniques to preprocess text and extract simplified phrases, significantly improving the system's ability to generate contextually appropriate queries.
  • Utilized Stanford NER to enhance entity recognition, enabling precise question formation by identifying and categorizing text-based entities effectively.
  • Addressed complex challenges in pronoun resolution and entity recognition by employing coreference resolution techniques, ensuring questions maintain clarity and relevance to the input text.
  • Designed a robust algorithm for generating both boolean and WH-questions, increasing the versatility of the question generator to accommodate a wider range of testing scenarios.
  • Integrated a hybrid entity recognition system combining a 4-class and 7-class Stanford NER model to improve the breadth and accuracy of entity detection.
  • Devised methods to handle grammatical and logical inconsistencies in generated questions, such as ungrammatical constructions and incorrect WH-word usage, enhancing the quality and usability of the outputs.
  • Developed a logistic regression model to rank generated questions based on relevance and grammaticality, ensuring the highest quality questions are utilized for bot evaluation.
  • Contributed to significant advancements in automated cognitive assessment tools, facilitating efficient and scalable bot training and evaluation processes.

Shortest Job First (SJF) Algorithm

  • Implemented the Shortest Job First scheduling algorithm, which selects the process with the smallest execution time to run next, effectively minimizing overall waiting time and increasing system efficiency.
  • Developed the project in a simulated environment using C++ to model process scheduling behavior, providing a hands-on understanding of preemptive and non-preemptive scheduling methods.
  • Addressed the challenge of predicting the length of the next CPU burst by employing historical data, enhancing the accuracy of the next CPU burst predictions through regression analysis and historical averages.
  • Conducted a comparative analysis with other scheduling algorithms like First-Come, First-Served (FCFS) and Round Robin (RR), demonstrating the conditions under which SJF provides optimal performance in terms of throughput and response time.
  • Enhanced the algorithm's practical applicability by incorporating it into a custom-developed operating system simulation, showcasing its effectiveness in real-world scenarios within a controlled environment.
